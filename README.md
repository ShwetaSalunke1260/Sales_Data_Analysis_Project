# Sales Data Analysis & Customer Purchase Prediction

## Project Overview
This project analyzes retail transaction data to extract business insights and predict repeat customer behavior using Machine Learning.

## Dataset
Public Online Retail Dataset from UCI Machine Learning Repository  
Download Link:  
https://archive.ics.uci.edu/ml/datasets/Online+Retail

*(Dataset not included in repo due to GitHub file size limit)*

## Technologies Used
- Python (Pandas, NumPy, Matplotlib)
- SQL (MySQL)
- Machine Learning (Scikit-learn)
- Jupyter Notebook

## Key Features
- Data cleaning and preprocessing  
- Exploratory Data Analysis (EDA)  
- Monthly sales trend visualization  
- Top-selling product analysis  
- Repeat customer prediction (~83% accuracy)

## How to Run
1. Download dataset from the above link  
2. Place `online_retail.csv` in data folder  
3. Run SQL queries from `retail_queries.sql`  
4. Run `Sales_Analysis.ipynb`
